Web que permite realizar busquedas a wikipedia y visualizar los diez resultados más destacados. 
Cada uno de los resultados de la busqueda posee un pequeño fragmento del artículo ademas de un link hacia su ubicación en wikipedia.
Tambien se permite obtener un articulo de manera aleatoria.

Tecnologias utilizadas: Html, Css, Javascript, JQuery, Bootstraps, Ajax, API de Wikipedia.

Link: http://codepen.io/Juancard/full/YwvqXK/
